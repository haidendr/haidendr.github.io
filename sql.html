Mining insights from structured data with SQL

This GitHub repository features a collection of SQL and Python projects focused on data analysis, database querying, and applied data management. The first project delivers an in-depth analysis of U.S. public pension plans using Python and SQL, uncovering insights into funding trends and fiscal health across states. The second project offers a suite of MySQL practice scripts built around the “My Guitar Shop” database, emphasizing SQL fundamentals through real-world business scenarios. The third and fourth projects explore analytical SQL techniques using the `chickweight` dataset—one in MySQL and the other in PostgreSQL—demonstrating the use of aggregate, and percentile functions to derive statistical insights. Together, these projects showcase strong skills in SQL, data analysis, and database design across multiple platforms.

Public Pension Plan Analysis with SQL

This project showcases a comprehensive analysis of U.S. public pension plans using SQL and Python, leveraging data from the Public Plans Database (PPD) spanning 2001 to 2017. Through a Jupyter Notebook powered by pandas and SQLAlchemy, it explores funding trends, contribution compliance, and investment performance across state retirement systems. Key steps include data validation, auditing for completeness, ranking states by funding health, and evaluating long-term returns. The project highlights SQL proficiency and data analysis skills while uncovering insights into fiscal sustainability and reporting inconsistencies among public pension plans.

SQL using Guitar Shop Database

This project features a curated set of MySQL practice scripts built around the sample "My Guitar Shop" database. Designed for learning and skill-building, the scripts cover essential SQL concepts such as joins, aggregations, subqueries, and user privilege management. Each script demonstrates practical scenarios like identifying top discounted products, analyzing high-value orders, auditing unused categories, and summarizing purchases by product and category. This project offers hands-on experience with data analysis, reporting, and database administration in a structured, real-world context.

Aggregate functions - Chick weight dataset

This project showcases how to use aggregate functions in MySQL with the `chickweight` dataset, which tracks chick weights over time across different diets. It includes SQL operations like previewing data, calculating totals, averages, minimums, maximums, and counts using `UNION`, and applying the `ROW_NUMBER()` window function to rank records by weight.

Summary Statistics - Chick weight dataset

This project provides a set of PostgreSQL queries to generate comprehensive summary statistics from the `public.chickweight` table, which records chick weights over time under various diets. It calculates key aggregates—such as totals, averages, minimums, maximums, and counts—alongside percentile-based metrics like quartiles and medians using `PERCENTILE_CONT()`. The script employs SQL techniques like `UNION`, aggregate functions, and formatting with `ROUND()` to produce a unified statistical overview. Designed for PostgreSQL users, it serves as a practical tool for analyzing longitudinal weight data.
